# Velveteen SOC Lite

Velveteen SOC Lite is a lightweight PowerShell toolkit for Windows endpoint triage and host monitoring.

## Features
- Real-time suspicious process alerts
- Process-to-network correlation
- Persistence detection (services, startup, tasks)
- Driver inspection and validation
- Post-boot baseline logging

## Usage
See docs/quick-start.md

## Safety
This toolkit is read-only by default. It does NOT remove or modify system components.
