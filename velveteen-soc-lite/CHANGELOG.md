# Changelog

## v1.0.0 - Initial Release

### Added
- Velveteen-ProcessSentinel.ps1 (real-time process monitoring)
- Velveteen-NetTrace.ps1 (process-to-network correlation)
- Velveteen-PersistenceScan.ps1 (startup, services, tasks, WMI)
- Velveteen-DriverInspector.ps1 (driver enumeration + signature checks)
- Velveteen-BootBaseline.ps1 (post-boot system snapshot)
- Velveteen-Summary.ps1 (risk scoring + TXT/HTML output)

### Features
- SOC-style investigation workflow
- Severity classification (HIGH / SUSPICIOUS / REVIEW)
- Scoring model to reduce alert fatigue
- Automatic TXT report generation
- HTML dashboard output
- Sample output included

### Notes
- Toolkit is read-only
- Designed for educational and defensive use
